Static

      

          Default, element resides in the normal page flow.
      

    


    

      
Relative

      

          Element remains in the flow but can be nudged to left/right/top/bottom using css.
      

    


    

      
Relative

      
    


    

      
Absolute

      

          Element no longer in normal flow, you have absolute control over the position of the element.
      

    


    

      
Absolute

      
    


    

      
Absolute

      
      

    


    

      
Fixed

      

          Same as absolute, but element is positioned relative to browser, not document. Remains in place even in page scrolls.
      

    


    

      
Inherit

      

          Explicitaly inherit positioning from parent element; positioning doesn't cascade.
